# Ref'Em All

Ref'em all is an exercise to SEO and webmarketing at Ynov School. Made by Philippe Gambier and Baptiste Crouzet.

## Technologies

* Gulp 4
* Boostrap 4 Grid only
* Jquery 3.5
* Scroll Reveal
* SCSS
